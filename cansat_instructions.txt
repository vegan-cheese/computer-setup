To flash micropython onto the ESP32:
In Thonny, click Run > Configure Interpreter
Choose Micropython (ESP32) in the first box
Then "Install or update micropython (esptool)"
    target port is probably "/dev/ttyUSB0" but it might detect automatically
    family = ESP32-S3
    variant = Espressif - ESP32-S3




To run micropython on the ESP32:

In Thonny, click Run > Configure Interpreter
Choose Micropython (ESP32) in the first box
Port should be detected automatically but if not, it is likely to be "/dev/ttyUSB0"
From then onwards, just click run from within thonny




To install python modules on the ESP32:

For packages on PyPI:
In Thonny, click Tools > Manage Packages
Search for the name of the package and click install

For python files on GitHub:
sudo ampy put "filename" --port /dev/ttyUSB0
