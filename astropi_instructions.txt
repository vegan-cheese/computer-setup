To add packages to the virtual environment:

cd ~/Programming/AstroPi
./env/bin/pip install name_of_package




To run the code from within Thonny:

Click Run > Configure Interpreter
Set the python executable directory to be "/home/user/Programming/AstroPi/env/bin/python3"
Then just click run from within thonny




To use the replay simulation tool:

cd ~/Programming/AstroPi
Astro-Pi-Replay run main.py (I think that the file has to be called main.py for this to work)
